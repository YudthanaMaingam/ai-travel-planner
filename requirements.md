สร้าง web app สำหรับ วางแผนการท่องเที่ยวโดย AI (Gemini or ollama) และมีระบบนำทางใน map ตามที่ AI วางแผนไว้

Use Case
1. user สามารถพูดคุยกับ AI เพื่อให้ AI วางแผนการท่องเที่ยวได้ หรือให้ AI แนะนำสถานที่ท่องเที่ยวพร้อมวางแผนให้ เช่น ต้องการไปเที่ยวจังหวัดลำพูน 3วัน 2คืน AI ก็จะทำการสืบค้นข้อมูลว่าสถานที่ไหนน่าเที่ยวบ้างหรือเป็น landmark แล้วนำมาวางแผนการท่องเที่ยวให้กับ user
2. user สามารถบันทึกแผนการท่องเที่ยวได้ เพื่อมาเปิดย้อนดูทีหลังได้

เครื่องมือ
1. ใช้ Next.js เป็นหลัก
2. ใช้ map library ของ mapcn คู่มือ https://www.mapcn.dev/docs
3. database ใช้ mongodb (local mongodb community server)
4. อื่นๆ ตามที่เห็นสมควร
